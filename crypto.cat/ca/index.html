Cryptocat et permet xatejar amb privacitat.

Estem desenvolupant una aplicació web que encrypt de forma fàcil les tees converses. És de codi obert, està disposable en 32 llengúes i ja saps com fer-lo server.

D

Missatgeria encriptada pot ser accessible.

_

Cryptocat és software lliure que lluita per oferir-te un entorn obert i accessible per la missatgeria instantania que function en el teu navigator.

C

Cryptocat és un experiment de codi obert - l'objectiu és proportioner de la forma més fàcil i accessible per xatejar mentre es manté la sega privacitat en línia, perquè creiem en la privacitat a Internet.

És útil per a tots.

[

Els amics fan server Cryptocat per parlar sense revelar els missatges a tercers. Els activists de LGBTQ fan server Cryptocat per mantenir els missatges privats, realment privats. Els periodistes fan server Cryptocat per manterir les recerques i notícies confidencials. Crytocat està fet per a tothom.

Cryptocat es desenvolupa sota un principi de transparència radical. El nostre codi, el servitor de configuració, les operations de xarxa, les auditories, el finançament i tota la resta són públics per a la sega revisió, millora i control.
